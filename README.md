エンジニア情報記事共有コミュニティ[Zenn](https://zenn.dev/) に記事を公開するためのレポジトリ

自分のプロフィールは[こちら](https://zenn.dev/def_yuisato)

# Zenn CLI

記事作成

```shell
$ npx zenn new:article
```

プレビュー

```shell
$ npx zenn preview
```

プレビューの限定 URL を deploy する

```shell
$ ./preview-deploy.sh
```
